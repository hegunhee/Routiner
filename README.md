# Routiner
루티너는 습관을 등록하고 나중에 확인할 수 있는 앱입니다.  
## 계획  
1. DrawerLayout 등록  
    * 오늘의 루틴, 기록  
2. 오늘의 데일리 루틴 등록기능  
    * SharedPreference (날짜 저장)  
    * Room  
3. 기록 탭  
4. Core Routine(매일 자동으로 등록되는 루틴)  
5. Google Login(Firebase)  
6. Category 설정  
    * 공부, 운동 이런 카테고리들을 설정
    * 만약 된다면 db 버전을 올려야함
